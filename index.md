# Mint 개인정보처리방침

최종 업데이트: 2026-07-01

Mint(이하 "앱")는 개인 개발자가 만든 가계부 공유 앱입니다. 본 방침은 앱 이용 시 수집·처리되는 정보를 설명합니다.

## 1. 수집하는 정보

- 거래 내역 (금액, 카테고리, 메모, 날짜, 결제 수단)
- 예산 설정 정보
- 프로필 정보 (표시 이름, 프로필 사진)
  - 표시 이름은 기본값으로 기기 이름(예: "OOO의 iPhone")을 제안하며, 사용자가 언제든 직접 수정할 수 있습니다.
- 푸시 알림 발송을 위한 디바이스 토큰

## 2. 정보 저장 및 처리 방식

앱이 수집하는 모든 데이터는 **사용자 본인의 Apple iCloud 계정(CloudKit)**에 저장됩니다. 개발자가 별도로 운영하는 서버는 존재하지 않으며, 개발자를 포함한 어떤 제3자도 사용자의 데이터에 접근할 수 없습니다.

"공유방" 기능으로 가계부를 다른 사용자와 공유하는 경우, 해당 데이터는 사용자가 직접 초대한 구성원에게만 Apple CloudKit Sharing을 통해 공유됩니다.

## 3. 제3자 공유 및 트래킹

앱은 광고, 분석(Analytics), 트래킹 SDK 등 어떠한 제3자 서비스도 사용하지 않습니다. 사용자의 데이터를 제3자에게 판매·제공하지 않으며, 사용자를 다른 앱이나 웹사이트에서 추적하지 않습니다.

## 4. 푸시 알림

앱은 거래 추가/수정/삭제, 예산 초과 등을 알리기 위해 Apple Push Notification service(APNs)를 사용합니다. 알림은 기기 설정에서 언제든지 끌 수 있습니다.

## 5. 데이터 삭제

- 앱 삭제 시 기기에 저장된 데이터가 제거됩니다.
- 공유방에서 나가거나 공유방을 삭제하면 관련 공유 데이터도 함께 정리됩니다.
- iCloud 데이터 자체를 삭제하려면 기기의 iCloud 설정에서 앱 데이터 접근 권한을 해제하거나 직접 삭제할 수 있습니다.

## 6. 아동의 개인정보

본 앱은 만 13세 미만 아동을 대상으로 하지 않으며, 아동으로부터 고의로 개인정보를 수집하지 않습니다.
